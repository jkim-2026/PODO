### 🎥 RTSP Streaming Setup

RTSP 스트리밍 환경을 구축하기 위한 폴더입니다. PCB 이미지를 기반으로 가상 컨베이어 벨트 영상을 생성하고, 이를 RTSP 서버를 통해 송출하는 기능을 포함하고 있습니다.

#### 📂 주요 파일 설명

| 파일/폴더명 | 설명 |
| :--- | :--- |
| **`images/`** | 영상 생성에 사용되는 원본 PCB 이미지들이 들어있는 폴더입니다. |
| **`pcb_video.py`** | `images/` 폴더의 이미지를 사용하여 컨베이어 벨트 이동 영상을 생성하는 Python 스크립트입니다. 그림자, 비네팅 등의 효과가 포함된 프리미엄 영상을 제작합니다. |
| **`video_optimize.sh`** | 생성된 영상을 RTSP 스트리밍에 최적화된 형식(H.264)으로 재인코딩하는 쉘 스크립트입니다. |
| **`stream.sh`** | `ffmpeg`와 `pm2`를 사용하여 영상을 RTSP 주소로 송출하는 스크립트입니다. |
| **`mediamtx.yml`** | RTSP 서버인 `mediamtx`의 설정 파일입니다. |
| **`mediamtx`** | RTSP/RTMP/HLS 서버 실행 파일입니다. (바이너리) |
| **`*.mp4`** | `pcb_video.py` 또는 `video_optimize.sh`를 통해 생성된 영상 파일들입니다. |

#### ⚠️ 주의사항
- 이미지, 영상 파일(`*.mp4`)과 `mediamtx` 바이너리는 용량 문제로 Git에 포함되지 않았습니다.
- 테스트가 필요하신 분은 AWS의 `~/rtsp` 경로에서 파일을 복사해 사용하시거나 별도로 요청해 주세요.
